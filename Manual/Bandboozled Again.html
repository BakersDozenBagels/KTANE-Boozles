Words have been Caesar shifted forward A steps [0,36) and rotated B steps left [0,10) (0 is after Z)

Encrypt corresponding word, buttons have exactly one letter match among them.

Its note value is how long to hold for (timer ticks)

You're always holding the button whose letter is first in Chinese reading order in the chart

You're starting the hold on timer's last digit = B

Once done, if A is odd, tap all brass buttons, otherwise all wooden (invert press if label has a loop (press <-> no press))

LED order is: correct button, correct start, correct length, correct presses
